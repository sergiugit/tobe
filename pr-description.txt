**Please describe the application briefly.**
Tobe is a privacy-focused YouTube client for desktop. It lets you browse, search, and watch YouTube videos without ads, tracking, or a Google account. It uses the InnerTube API directly (no third-party APIs or tracking scripts) and yt-dlp for video streaming. Built with Tauri v2 (Rust backend, vanilla JS frontend). Source code: https://github.com/sergiugit/tobe
